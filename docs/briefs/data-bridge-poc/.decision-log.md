# Decision Log — Index

## 2026-06-05

- **Scope POC limité à 2 sources** : décision de simplicité, extensible post-POC
- **Profil cible = non-technique uniquement** : le profil data analyst est explicitement hors scope POC
- **Pas de connexion directe à une BDD** : les exports couvrent ce besoin sans la complexité d'une connexion
- **L'outil est un intermédiaire** : il produit un dataset unifié, pas de rapports ou visualisations
- **LLM local obligatoire** : contrainte de confidentialité des données pour le marché des administrations françaises, pas de données sortant de la machine
- **Différenciation clé** : IA propose le mapping + explique en langage naturel, l'humain valide — inverse du paradigme des outils existants
- **Question ouverte : fiabilité du matching** — pas de solution arrêtée, à traiter en phase design/archi
